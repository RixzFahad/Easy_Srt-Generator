# 🎬 SubtitleGen  

SubtitleGen is a simple web-based **subtitle generator** that helps content creators quickly turn scripts into **SRT files**.  
It’s designed for YouTubers, Instagram creators, and Facebook video editors who want to **boost engagement with captions**.  

---

## 🚀 Features
- 📝 Paste your script and generate subtitles instantly.  
- ⚡ Choose subtitle split modes (auto, sentence, word).  
- ⏱️ Set custom duration, gaps, and start times.  
- 📂 Instant **SRT preview** and download.  
- 🌍 Supports **countries & languages section** for multilingual subtitle creation.  
- ✨ Responsive modern UI with animations and background slideshow.
