# 🎬 Easy SRT Generator

A simple and powerful **Python-based subtitle generator** that automatically creates `.srt` subtitle files from text or transcripts.

This tool helps creators quickly generate subtitles for **YouTube videos, reels, podcasts, online courses, and social media content**.

---

## 🚀 Project Overview

Creating subtitles manually can be time-consuming.  
**Easy SRT Generator** simplifies this process by automatically converting text into a properly formatted **SRT subtitle file** with timestamps.

This tool is useful for:

- 🎥 YouTube creators  
- 📱 Instagram / Reels creators  
- 🎙 Podcast creators  
- 📚 Online educators  
- 🧑‍💻 Developers working with video processing

## 🧠 How It Works

The generator processes text input and converts it into the **standard SRT subtitle format**.

Each subtitle block contains:

1. Subtitle number  
2. Start timestamp → End timestamp  
3. Subtitle text  

Example SRT output:

```
1
00:00:01,000 --> 00:00:03,000
Hello everyone, welcome to my channel.

2
00:00:03,500 --> 00:00:06,000
Today we are going to learn how subtitles work.
```
